Полная документация проекта разделена на несколько разделов для удобства навигации.

## Структура документации

- **[API.md](API.md)** — Все API эндпоинты, их параметры и примеры ответов
- **[FINANCIAL_LOGIC.md](FINANCIAL_LOGIC.md)** — Финансовая логика: грейс-период, лимиты, платежи, бюджеты
- **[CELERY_TASKS.md](CELERY_TASKS.md)** — Планировщик задач (Celery Beat): расписание и логика

- **[CONFIGURATION.md](CONFIGURATION.md)** — Переменные окружения и настройка параметров

## Общая архитектура

Система состоит из:
- **FastAPI Backend** — основной сервис с REST API
- **PostgreSQL** — хранилище данных
- **Ollama** — локальная LLM для парсинга СМС
- **Celery** — фоновые задачи (расчёты, уведомления)
- **Redis** — брокер сообщений для Celery
- **Nginx** — прокси и точка входа
